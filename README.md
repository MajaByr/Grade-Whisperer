## Overview

Grade Whisperer listens to the signals hidden within student performance data, combining statistical inference with predictive modeling to understand what drives academic success.

This project provides:

* In-depth analysis of the [UCI Student Performance](https://archive.ics.uci.edu/dataset/320/student+performance) dataset
* Regression and classification tasks derived from educational outcomes
* Comparison of linear, regularized, tree-based and ensemble models
* Predictor importance ranking and effect interpretation
* Investigation of linear and nonlinear relationships
* Evaluation of model generalization and predictive accuracy
* Reproducible statistical learning workflow
